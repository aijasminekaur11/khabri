## Adding Things

When Jasmine says "Add [something]":
- Give the edit link
- Give EXACT yaml/text to paste
- Tell her WHERE to paste (which section)

Example - "Add Hindu Business Line":
Give link to sources.yaml, give this to paste:
```yaml

    url: "https://www.thehindubusinessline.com/real-estate/"
    enabled: true
    frequency_minutes: 60
```

## Removing Things

When Jasmine says "Remove [something]" or "Stop tracking [something]":
- Give the edit link
- Tell her to find the item
- Option 1: Delete the entire block
- Option 2: Change `enabled: true` to `enabled: false` (recommended - can undo later)

## Changing Schedule Time

Morning digest is at 7 AM IST. To change:
- Link: scheduled-digest.yml
- Find: `cron: '30 1 * * *'` (this is 7 AM IST)
- IST to UTC: Subtract 5 hours 30 minutes

Common times:
- 6 AM IST = `'30 0 * * *'`
- 7 AM IST = `'30 1 * * *'`
- 8 AM IST = `'30 2 * * *'`
- 9 AM IST = `'30 3 * * *'`

Evening:
- 4 PM IST = `'30 10 * * *'`
- 5 PM IST = `'30 11 * * *'`
- 6 PM IST = `'30 12 * * *'`

## Adding Events (like RBI meeting)

For new events, add to schedules.yaml under `events:` section:
```yaml
  - name: "Event Name"
    date: "2026-MM-DD"
    start_time: "HH:MM"
    end_time: "HH:MM"
    enabled: true
    keywords:
      - "keyword1"
      - "keyword2"
```

Note: For event to have automatic alerts like Budget, a workflow file is also needed. Tell Jasmine you'll need developer help to create the GitHub Action workflow for new events.
